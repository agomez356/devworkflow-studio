# WebApp Example

React + TypeScript webapp demonstrating DevWorkflow Studio MCP integration.

## Quick Start

### Local
```bash
npm install
npm run dev
```

### Docker
```bash
docker compose up
```

App: `http://localhost:5173`

## Features

- ✅ React 18 + TypeScript
- ✅ Vite dev server with HMR
- ✅ User management UI
- ✅ Docker multi-stage build
- ✅ MCP server integration

## MCP Integration

Uses DevWorkflow Studio MCP servers for:
- Code quality (lint/format)
- Git workflow automation
- Documentation generation

See `AGENTS.md` for details.
